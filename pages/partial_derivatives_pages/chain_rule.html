Chapter 2.5 : Chain Rule



For the chain Rule, constructing tree diagrams gives us a pathway for the chain rule in any situation.

Here is an example of the following Chain Rule:

We initiate by concentrating on the function and then expanding it outwardly. The primary set of branches portrays the variables in the function, and from each endpoint, we depict further components that show the variables dependent on both "\(x\)" and "\(y\)."

Each letter is linked by a line, representing a partial derivative, and the letter in the numerator of the partial derivative denotes the tree's upper "node."

In contrast, the letter in the denominator indicates the lower "node." To apply the chain rule, we start from the bottom and ascend the tree for each branch that concludes with the variable we need to take the derivative concerning (in this instance, "\(s\)").

Then, along that set of branches, we multiply the derivatives. Lastly, we repeat this process for each branch ending at "\(s\)" to sum up the outcomes to obtain the chain rule for that specific situation.
