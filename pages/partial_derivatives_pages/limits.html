Chapter 2.1 : Limits

This section focuses on limits that involve functions of multiple variables. Therefore, the discussion will primarily concentrate on the limits of functions with two variables.

However, the concepts presented can be extended to functions with more than two variables.

Before delving into this topic, briefly review how the limits of functions of one variable work.

Provided,

\(\mathop {\lim }\limits_{x \to {a^ + }} f\left( x \right) = \mathop {\lim }\limits_{x \to {a^ - }} f\left( x \right) = L\)

Also, recall that,

\(\mathop {\lim }\limits_{x \to {a^ + }} f\left( x \right)\)

This is a right-hand limit, meaning only values of \(x\) greater than a are considered.

As for the left-hand limit, only values of \(x\) less than \(a\) are considered.

\(\mathop {\lim }\limits_{x \to {a^ - }} f\left( x \right)\)

These limits are typically used when dealing with non-continuous functions where each side of the discontinuity displays a different limit.

For multi-variables, we can have two variables converge at different values.

For example:

\(\mathop {\lim }\limits_{x \to a\atop y \to b} f\left( {x,y} \right)\hspace{0.5in}\mathop {\lim }\limits_{\left( {x,y} \right) \to \left( {a,b} \right)} f\left( {x,y} \right)\)

For simplicity's sake, we will use the 2nd notation more often than the 1st.

Note that in the 2nd notation, we are trying to determine what \(f\left(x,y \right)\) is doing as we move \(\left(x,y \right)\) towards \(\left(a,b \right)\) without actually letting \(\left(x,y \right)=\left(a,b \right)\).

The notation used here needs to be revised. A limit must be well-defined so that the function produces the same outcome regardless of the approach towards the point \(\left(a, b \right)\).

In short, we need to check infinite paths to ensure the function has the same output regardless of the path we use to approach \(\left(a,b \right)\).

So let's use one of the main ideas from Calculus 1 to help us:

\(f \left(x,y \right)\) is

continuous

at the point \(\left(a,b \right)\) if,

\(\mathop {\lim }\limits_{\left( {x,y} \right) \to \left( {a,b} \right)} f\left( {x,y} \right) = f\left( {a,b} \right)\)

If a function is continuous at a point, we can evaluate the limit of the function at that point by simply substituting the point into the function.

\(\mathop {\lim }\limits_{\left( {x,y} \right) \to \left( {a,b} \right)} f\left( {x,y} \right) = f\left( {a,b} \right)\)

Even for multivariable functions, the standard functions known to be continuous remain continuous as long as we avoid dividing by zero, taking square roots of negative numbers, or taking logarithms of zero or negative numbers.

However, it is essential to remember the concept of paths, which can help us determine if a limit does not exist. By finding two paths that lead to different values as we approach the point, we can conclude that the limit does not exist.
