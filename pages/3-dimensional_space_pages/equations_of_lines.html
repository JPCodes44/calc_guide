Chapter 1 : 3-Dimensional Space

In this section, we will be learning how to make a line in 3D as opposed to 2D.

Suppose we have a vector function:

\(\vec r\left( t \right) = \left\langle {t,2} \right\rangle\)

The vector function above is a function can take one or more variables, however if the amount of variables are more than two, the graph will cease to exist in space. In other words, a vector function returns a vector in the given dimension (In the example above it is \(\mathbb{R}^2\), due to having 2 slots in the vector brackets).

Lets determine the graph of the vector function. In order to do this, we will need to find the position vectors on the graph of the vector functions. We can do this by plugging in numbers which will return the points/position vectors of our vector function.

Given the position vector: \(\vec v\left( t \right) = \left\langle {a,b} \right\rangle\) where the vector starts at the origin and ends at \(\left( {a,b} \right)\) we can plug in some values within the vector function then we can just plot the points.

\(\vec r\left( -1 \right) = \left\langle {-1,2} \right\rangle\)

\(\vec r\left( 1 \right) = \left\langle {1,2} \right\rangle\)

\(\vec r\left( 3 \right) = \left\langle {3,2} \right\rangle\)

\(\vec r\left( -2 \right) = \left\langle {-2,2} \right\rangle\)

Now we can plot the following points:

\(\left( {-1,2} \right)\)

\(\left( {1,2} \right)\)

\(\left( {3,2} \right)\)

\(\left( {-2,2} \right)\)

If we plot all of these points we get the sketch:

The position vector in grey is seen for various values of t, where the graph of the vector equation is indeed \(y=1\). Note that vector functions do not have to be lines, they can also be elipses, triangles etc..

Back on the subject of line equations, in order to write an equation in \(\mathbb{R}^3\), the slope m is not a simple number unlike in \(\mathbb{R}^2\) so that means a line contains a three dimensional slope. However it is required that we have something that will describe direction in 3 dimensions which is where vectors come in.

Lets suppose we know a point on the line \({P_0}\) = \(\left( {{x_0},{y_0},{z_0}} \right)\) where \(\vec v \) = \(\left\langle {a,b,c} \right\rangle\) (Note: \(\vec v \) does not have to be on the line, only parallel) and \({P}\) = \(\left( {{x},{y},{z}} \right)\) is any point on the line.

Since the slope in \(\mathbb{R}^3\) is a vector, we can use the two points on the line as poition vectors. We will let \(\overrightarrow {{r_0}}\) and \(\overrightarrow {{r}}\) be position vectors for \({P_0}\) and \({P}\)
