The domains of functions containing more than one variable would have to be intervals in terms of \(x\) and \(y\) instead of a single variable in \(\mathbb{R}^1\).

The following paragraph discusses the concept of level curves, also known as contour curves, which are two-dimensional curves obtained by setting a function equal to a constant value. For example, if the function is given by \(z=f(x,y)\), then the equation of the level curves is \(f(x,y)=k\), where k is any constant number.

In some cases, the equation may be in the form \(f(x,y,z)=0\), and in these situations, the equation of the level curves is \(f(x,y,k)=0\). An example of level curves is the elevation map of a piece of land, which gives the contour curves for the function representing the elevation in the area.

Let's do a quick example of this.

Example 1

Determine the domain of each of the given functions.

  1. \(f \left(x,y \right) = \dfrac{x}{y}\)
  2. \(f \left(x,y \right) = \sqrt{x} + y\)
  3. \(f \left(x,y \right) = \ln \left(3 - x^2 - 3y^2 \right)\)

(a) \(f \left(x,y \right) = \dfrac{x}{y}\)

Show Solution

It is clear that y cannot be 0 because or else the function would be undefined so:

\(y \neq 0\)


(b) \(f \left(x,y \right) = \sqrt{x} + y\)

Show Solution

It is clear that x cannot be less than 0 because or else the function would be undefined so:

\(x \geq 0\)


(c) \(f \left(x,y \right) = \ln \left(3 - x^2 - 3y^2 \right)\)

Show Solution

We should note that we cannot take the natural log of a negative number so we have the following domain:

\(3 - {x^2} - 3{y^2} > 0\hspace{0.25in} \Rightarrow \hspace{0.25in}\,\,\,\frac{{{x^2}}}{3} + {y^2} < 1\)

Example 2

Determine the domain of the given function.

\(f\left( {x,y,z} \right) = \dfrac{1}{{\sqrt {{x^2} + {y^2} + {z^2} - 8} }}\)

Show Solution

We should note that we cannot take the square root of a negative number so we have the following domain:


\({x^2} + {y^2} + {z^2} - 8 > 0\hspace{0.25in}\,\,\,\,\, \Rightarrow \hspace{0.25in}\,\,\,{x^2} + {y^2} + {z^2} > 8\)


The domain of a function is the set of all possible input values for which the function is defined. In this case, the function is defined only for input points outside a sphere with a radius of \(\sqrt{8}\) and centred at the origin. So, the function's domain consists of all points in space that are not inside this sphere.

Imagine contours as the intersections of the surface moving through the surface as the \(k\) value is increasing/decreasing.

We will look at level surfaces for functions containing three variables, where \(k = f(x,y,z)\) and k can include any number.

This last section introduces the concept of traces, which are curves that represent the intersection of a surface and a plane given by \(x=a\) or \(y=b\). Traces are similar to contours, which are the intersection of a surface with a plane \(z=k\), as explained earlier. In other words, contours and traces represent the intersection of a surface with a plane, but contours are obtained by varying the value of \(z\), while traces are obtained by changing the value of \(x\) or \(y\).

Let's take a quick look at an example of traces.
