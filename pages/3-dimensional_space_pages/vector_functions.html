Chapter 1.5 : Vector Functions

Vector functions were previously introduced in writing the equation of a line in \(\mathbb{R}^3\) using a vector function. However, this section aims to examine vector functions further and explore their use beyond lines in \(\mathbb{R}^3\).

Before we get into graphing and identifying vector functions, we need to talk about the domain of a vector function. The domain of a vector function is the interval of all \(t\)'s for which all the vector component functions are defined.

Let's explore the graph of vector functions. To graph a vector function, we can treat the vector returned by the function as a position vector for points on the graph. This means that a position vector, such as \(v = \left\langle {a,b,c} \right\rangle\), starts at the origin and ends at the point \((a,b,c)\).

To sketch the graph of a vector function, we can input values of t into the function and plot the resulting position vectors as points on the graph.

Recall the ellipse vector function \(r(t)= \left\langle{6 cost, 3sint}\right\rangle\). Notice that the \(x\) component represents \(6cost\) and the \(y\) component represents \(3sint\). So if we make an equation for each of these components, we get the following:

\(x = 6\cos t\hspace{0.25in}\hspace{0.25in}y = 3\sin t\)

These concepts look similar to parametric equations, as we saw in previous calculus courses.

This is a fundamental concept in the analysis of vector functions. Representing any vector function using parametric equations defining the same graph is possible. Before diving into vector functions, Obtaining a particular formula necessary for starting vector functions would be beneficial.

As mentioned earlier in this section, having vector functions with two variables is possible. In such cases, the graphs of these vector functions will be surfaces. To avoid forgetting this fact, let's work through an example that involves a vector function with two variables.

For the final topic of this section, let's focus on parameterizing vector functions. Given any function of one or two variables, we can always write the vector form in terms of just one or two variables like so:

\(\vec r\left( x \right) = x\,\vec i + f\left( x \right)\,\vec j\hspace{0.25in}\hspace{0.25in}\hspace{0.25in}\vec r\left( y \right) = h\left( y \right)\,\vec i + y\,\vec j\)

Or

\(\begin{array}{c}\vec r\left( {x,y} \right) = x\,\vec i + y{\kern 1pt} \vec j + g\left( {x,y} \right)\,\vec k\hspace{0.25in}\hspace{0.25in}\hspace{0.25in}\vec r\left( {y,z} \right) = g\left( {y,z} \right)\,\vec i + y{\kern 1pt} \vec j + z\,\vec k\\ \vec r\left( {x,z} \right) = x\,\vec i + g\left( {x,z} \right){\kern 1pt} \vec j + z\,\vec k\end{array}\)


This will be a crucial idea in calculus three, as fluency in parametric equations is necessary for surface and line integrals.
