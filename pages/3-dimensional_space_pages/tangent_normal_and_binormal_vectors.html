Chapter 1.7 : Tangent, Normal and Binormal Vectors

In this section, we will explore using derivatives in vector functions, which have multiple applications that ultimately rely on the first application.

Previously, we used the concept that a function's derivative represents the tangent line's slope. We arrive at the same result in vector functions except for one detail.

When dealing with a vector function, \(r(t)\), we define \(r'(t)\) as the tangent vector, provided that it exists and is not equal to zero. The tangent line to \(r(t)\) at point \(P\) is the line that passes through \(P\) and is parallel to the tangent vector, \(r'(t)\). It's important to note that the requirement \(r'(t) \neq 0\) must have a tangent vector.

Let's suppose we had:

\(\vec r'\left( t \right) = \vec 0\)

The following vector has no magnitude, so it would be impossible to determine the direction of the tangent.

However, if \(r(t) \neq 0\), the unit tangent vector to the curve is given by,

\(\vec T\left( t \right) = \dfrac{{\vec r'\left( t \right)}}{{\left\| {\vec r'\left( t \right)} \right\|}}\)

Let's give an example:

Example 1

Find the general formula for the unit tangent vector and tangent vector of the curve given by

\(\vec r\left( t \right) = {t^3}\,\vec i + 3\sin t\,\vec j + 2\cos t\,\vec k\)

Show Solution

To find the tangent vector, we take the derivative of the curve:

\(\vec r'\left( t \right) = 3t^2\,\vec i + 3\cos t\,\vec j - 2\sin t\,\vec k\)

For the unit tangent vector, we will need the norm of the tangent vector:

\(\begin{align*}\left\| {\vec r'\left( t \right)} \right\| & = \sqrt {9{t^4} + 9{{\cos }^2}t + 4{{\sin }^2}t} \\ \\ & = \sqrt {9{t^2} + 13} \end{align*} \)

Which we can then plug into the unit tangent vector equation:

\(\begin{align*}\vec T\left( t \right) & = \dfrac{1}{{\sqrt {9{t^2} + 13} }}\left( {3{t^2}\,\vec i + 3\cos \,t\,\vec j - 2\sin t\,\vec k} \right)\\ \\ & = \frac{{2t}}{{\sqrt {9{t^2} + 13} }}\,\vec i + \frac{{2\cos t}}{{\sqrt {9{t^2} + 13} }}\vec j - \frac{{2\sin t}}{{\sqrt {9{t^2} + 13} }}\,\vec k\end{align*}\)

Let's also talk about the unit normal and binormal vectors.

The Unit normal vector is given by:

\(\vec N\left( t \right) = \dfrac{{\vec T'\left( t \right)}}{{\left\| {\vec T'\left( t \right)} \right\|}}\)

The unit normal is orthogonal to the unit tangent vector and the curve.


Fact:

Suppose that \(r(t)\) is a vector such that \(||r(t)|| = c\) for all \(t\). Then \(r'(t)\) is orthogonal to \(r(t)\).

Proof:

The definition of the unit normal vector shows that \(||T(t)|| = 1\) for all \(t\) since \(T(t)\) is normal to \(T(t)\). But since \(T(t)\) is tangent to the curve, \(T'(t)\) and its vector must be orthogonal. We divide by \(||T(t)||\) and obtain a unit normal vector.

Since the binormal vector is defined as the result of the cross product between the unit tangent vector and the normal unit vector, we can conclude that the binormal vector is perpendicular to both the tangent vector and the normal vector.
