Chapter 1.8 : Arc Length with Vector Functions

To determine the length of a vector function from \(a\le t \le b\), recall that the vector function in parametric form is \(x=f(t)\) \(y=g(t)\) \(z=h(t)\).

Then we can use the parametric functions to determine the arc length using the formula:

\(L = {\displaystyle\int_{{\,a}}^{{\,b}}}{{\sqrt {{{\left[ {f'\left( t \right)} \right]}^2} + {{\left[ {g'\left( t \right)} \right]}^2}} \,dt}}\)

We can simplify this further to:

\(\left\| {\vec r'\left( t \right)} \right\| = \sqrt {{{\left[ {f'\left( t \right)} \right]}^2} + {{\left[ {g'\left( t \right)} \right]}^2} + {{\left[ {h'\left( t \right)} \right]}^2}}\)

Therefore the arc length formula is:

\(L = {\displaystyle\int_{{\,a}}^{{\,b}}}{{\left\| {\vec r'\left( t \right)} \right\|\,dt}}\)

We can also define the arc length function as:

\(\left\| {\vec r'\left( t \right)} \right\| = \sqrt {{{\left[ {f'\left( t \right)} \right]}^2} + {{\left[ {g'\left( t \right)} \right]}^2} + {{\left[ {h'\left( t \right)} \right]}^2}}\)

Let's look at an example to see why this is important.

Example 2

Determine the arc length function for \(\vec r\left( t \right) = \left\langle {t,3\cos \left( {t} \right),4\cos \left( {2t} \right)} \right\rangle\)

Show Solution

From the previous example, we know that,

\(\left\| {\vec r'\left( t \right)} \right\| = 2\sqrt {10}\)

The arc function is then,

\(s\left( t \right) = {\displaystyle\int_{{\,0}}^{{\,t}}}{{2\sqrt {10} \,du}} = \left( {2\sqrt {10} \,u} \right)_0^t = 2\sqrt {10} \,t\)

We can re-parameterize the function in terms of \(s \left(t \right) \)by solving for \(t\) as follows:

\(t = \dfrac{s}{{2\sqrt {10} }}\)

What is the purpose?

We can now indicate how much distance we have traveled \(\left(s \right)\) along the curve. Also, notice that we start the distance measurement at \(t=0\).

Example 3

Where on the curve \(\vec r\left( t \right) = \left\langle {t,3\cos \left( {t} \right),4\cos \left( {2t} \right)} \right\rangle\) are we after traveling for a distance of \(\dfrac{\pi}{2}\)

Show Solution

\(\vec r\left( {t\left( s \right)} \right) = \left\langle {\dfrac{s}{{\sqrt {10} }},3\sin \left( {\dfrac{s}{{\sqrt {10} }}} \right),3\cos \left( {\dfrac{s}{{\sqrt {10} }}} \right)} \right\rangle\)

We will sub in \(\dfrac{\pi}{2}\) into the re-parametrized function to get our location:

\(\vec r\left( {t\left( {\dfrac{{\pi \sqrt {10} }}{3}} \right)} \right) = \left\langle {\dfrac{\pi }{3},3\sin \left( {\dfrac{\pi }{3}} \right),3\cos \left( {\dfrac{\pi }{3}} \right)} \right\rangle = \left\langle {\dfrac{\pi }{3},\dfrac{{3\sqrt 3 }}{2},\dfrac{3}{2}} \right\rangle\)
