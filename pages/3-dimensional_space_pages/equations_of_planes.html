Chapter 1.2 : Equations of Planes

In the last section, we looked at a few equations of planes that did not contain three variables. In this section, we will talk about deriving a more general equation for 3D planes.

Firstly, we know a point on the plane \(P_0=(x_0,y_0,z_0)\) and a normal vector orthogonal (90 degrees) to that same plane. So let's also set \(r_0\) and \(r\) as position vectors for \(P_0\) and \(P\).

Here is a graph of the following planes and vectors:

Notice how \(r-r_0\) forms a vector that is parallel and onto the plane. In linear algebra, we learned that the dot product of two orthogonal vectors is \(0\). This allows us to form a vector equation for the plane consisting of the normal vector and \(r-r_0\):

\(\vec n\centerdot \left( {\vec r - \overrightarrow {{r_0}} } \right) = 0\hspace{0.25in}\hspace{0.25in} \Rightarrow \hspace{0.25in}\hspace{0.25in}\vec n\centerdot \vec r = \vec n\centerdot \overrightarrow {{r_0}}\)

We will start with a vector equation in terms of \(a\), \(b\), and \(c\) and the starting/end \(x\),\(y\), and \(z\) values.

\(\begin{align*}\left\langle {a,b,c} \right\rangle \centerdot \left( {\left\langle {x,y,z} \right\rangle - \left\langle {{x_0},{y_0},{z_0}} \right\rangle } \right)& = 0\\ \left\langle {a,b,c} \right\rangle \centerdot \left\langle {x - {x_0},y - {y_0},z - {z_0}} \right\rangle & = 0\end{align*}\)

Computing the dot product, we get:

\(a\left( {x - {x_0}} \right) + b\left( {y - {y_0}} \right) + c\left( {z - {z_0}} \right) = 0\)

If given an equation of planes, we can easily find the normal vector by looking at the coefficients \(a\), \(b\), and \(c\).

Let's work some examples.

Example 1

Determine the equation of the plane containing the points \(P = \left(2,-1,3 \right)\), \(Q = \left(-1,4,1 \right)\) and \(R = \left(0,-2,5 \right)\) is .

Show Solution

To write down the equation of a plane, we need all of the required points we have. In addition, we also need to find a normal vector using the cross-product.

Let's form the following two vectors from the given points:


\(\overrightarrow {PQ} = \left\langle {2,3,4} \right\rangle \hspace{0.25in}\hspace{0.25in}\overrightarrow {PR} = \left\langle { - 1,1,2} \right\rangle\)


Notice that these points lie entirely in the plane since we used points from the plane to form those two vectors.

We also know that the cross-product of two vectors will be orthogonal to both of the two vectors. Therefore, we can use the cross-product of the two vectors to find our normal vector:


\(\vec n = \overrightarrow {PQ} \times \overrightarrow {PR} = \left| {\begin{array}{*{20}{c}}{\vec i}&{\vec j}&{\vec k}\\2&3&4\\{ - 1}&1&2\end{array}} \right|\,\,\,\,\begin{array}{*{20}{c}}{\vec i}&{\vec j}\\2&3\\{ - 1}&1\end{array} = 2\vec i - 8\vec j + 5\vec k\)


Now we can form the following equation of the plane:


\(\begin{align*}2\left( {x - 1} \right) - 8\left( {y + 2} \right) + 5\left( {z - 0} \right) & = 0\\ 2x - 8y + 5z & = 18\end{align*}\)


We could have also used Q or R for the plane equation.

Example 2

Determine if the plane given by \(- x + 2z = 10\) is orthogonal, parallel, or neither to the line \(\vec r = \left\langle {5,2 - t,10 + 4t} \right\rangle\)

Show Solution

Remember that we can pick off a vector normal to the plane by focusing on the coefficients? This will give us \(\vec n = \left\langle { - 1,0,2} \right\rangle\). we can also obtain a parallel vector, such as \(v = \left\langle {0, - 1,4} \right\rangle\). If n and v are parallel, then v is orthogonal to the plane and parallel to the line. Therefore, the plane will be orthogonal if the two vectors are parallel to the line.

To check, we will see if the cross-product is 0:


\(\vec n \times \vec v = \left| {\begin{array}{*{20}{c}}{\vec i}&{\vec j}&{\vec k}\\{ - 1}&0&2\\0&{ - 1}&4\end{array}} \right|\,\,\,\,\begin{array}{*{20}{c}}{\vec i}&{\vec j}\\{ - 1}&0\\0&{ - 1}\end{array} = 2\vec i + 4\vec j + \vec k \ne \vec 0\)


As we can see, the vectors are not parallel, which means the plane and the line are not orthogonal.

Now we need to check if the plane and the line are parallel; we will check by using the dot product for n and v as such:

\(\vec n\centerdot \vec v = 0 + 0 + 8 = 8 \ne 0\)
