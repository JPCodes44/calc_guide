Chapter 2.5: Lagrange Multipliers

Previously we only accounted for finding absolute extrema within the boundary but what about the interior of a function?

This section presents an alternative approach to optimize a function while considering a given constraint(s). These constraints could be equations that define the boundary of a region, but the focus of this section is on something other than such problems. The method described here only requires a general constraint and does not concern itself with the origin of the constraint.
The goal is to find the minimum and maximum value of a function \(f \left(x, y, z\right)\) while satisfying a constraint \(g \left(x, y, z \right) = k\). The constraint may or may not describe the boundary of a region. Although the process is relatively straightforward, it can still be daunting.

Let's first define the method of Lagrange Multipliers:

  1. Solve the following system of equations.

  2. Plug in the solutions, \(\left(x,y,z \right)\), from the first step into \(f \left(x,y,z \right)\) and identify the max and min values given that they exist and the gradient of the constraint does not equal \(0\) at \(\left(x,y,z \right)\).

The constant lambda \(\left(\lambda\ \right)\) is called the Lagrange multiplier.
We will solve for \(f_x\), \(f_y\), and \(f_z\) giving the following system of equations:

\(\left\langle {{f_x},{f_y},{f_z}} \right\rangle = \lambda \left\langle {{g_x},{g_y},{g_z}} \right\rangle = \left\langle {\lambda {g_x},\lambda {g_y},\lambda {g_z}} \right\rangle\)

\({f_x} = \lambda {g_x}\hspace{0.25in}{f_y} = \lambda {g_y}\hspace{0.25in}{f_z} = \lambda {g_z}\)

Note that in some cases, minimums and maximums will not exist in every case, so we will have to check for those first.

To provide a physical explanation for the equations above, let's examine the minimum and maximum values of the function \(f \left(x,y \right) = 8x^2 - 2y\) subject to the constraint \(x^2 + y^2 = 1\). In problem #2 of the practice exercises for this section, it will be shown that the minimum value of \(f \left(x,y \right)\) is \(-2\), which occurs at \(\left(0,1 \right)\), while the maximum value of \(f \left(x,y \right)\) is \(8.125\), which occurs at \(\left(-3 \sqrt{ \frac{7}{8} }, -1/8 \right)\) and \(\left(3\sqrt{\frac{7}{8}},-1/8 \right)\).

Here is a sketch of the following constraint and \(f \left(x,y \right) = k\) for various values of \(k\).

Note that solutions must lie somewhere on the graph of \(g\) (the constraint), which is \(x^2+y^2=1\).

From the given graphs, we can observe that the minimum or maximum values of the function \(f \left(x,y \right)\) occur when the graph of \(f \left(x,y \right) = k\) intersects with the constraint graph at a tangent point where their normal vectors are parallel. This is because the point of intersection must be on the constraint graph. To find the minimum or maximum value of \(f \left(x,y \right)\), we need to solve the system of equations, as mentioned earlier. The mathematical meaning is:

\(\nabla f\left( {x,y,z} \right) = \lambda \,\,\nabla g\left( {x,y,z} \right)\)

It is worth noting that while the above justification was explained in the context of a two-dimensional system, the same idea applies to higher dimensions where we work with surfaces instead of curves. In such cases, the points that give the minimum and maximum values of the surfaces would be parallel, and so would be their normal vectors.

In the above example, we did not calculate the values of \(\lambda\) as it is unnecessary to determine whether the point is a minimum or a maximum. Hence, in such problems, we often do not bother finding the value of \(\lambda\). On the other hand, in some cases, we may require the value of \(\lambda\) to solve the system, but we won't use it beyond finding the point.

In order to ensure that we have identified all possible locations of the highest and lowest points in a given problem, we must examine the endpoints of any variable ranges that exist. However, if our only concern is determining the actual values of these highest and lowest points, then examining the endpoints may not be necessary.

Let's take a look at another example.

In the examples above, the general formula for solving these problems comes as follows:

  1. Gather all of the equations needed to solve each variable.

  2. Find what each variable equals in each equation.

  3. Set the variables equal to the same value in the same equation.

  4. Solve for one of the unknowns.

  5. Repeat the process from step 4 for each possible case of equations.

  6. Write down all the possible extrema from all of the cases of equations.

  7. Plug in the points in f and determine the max and min values.

  8. Plug in the points for the boundaries into f as well.
