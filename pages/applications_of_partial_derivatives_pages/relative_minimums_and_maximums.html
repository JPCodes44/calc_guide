Chapter 3.2 : Relative Extrema

In this section, we will define the minimum and maximums functions and identify relative minimums and relative maximums for functions of two variables.

It should be noted that a relative minimum of a function is not necessarily the smallest value that the function can take but rather means that in a particular region around the point \(\left(a, b \right)\), the function will always be greater than or equal to \(f \left(a, b \right)\). However, the function can be smaller outside of that region. Similarly, a relative maximum only implies that around \(\left(a, b \right)\), the function will always be smaller than or equal to \(f \left(a,b \right)\), but it can be larger outside of that region.

When extending the concept of critical points to functions of two variables, a critical point of the function \(f \left(x \right)\) was a value x=c such that either \(f' \left(c \right) = 0\) or \(f' \left(c \right)\) doesn't exist. A similar definition exists for critical points of functions of two variables.

To prove the first part, suppose \(\nabla f=0\) and sub in the definition for each component.

Note that both components in the definition MUST be \(0\), or it is not a critical point.

Fact:
If the point \(\left(a,b \right)\) is relative extrema of the function \(f \left(x,y \right)\) and its first order derivatives of \(f \left(x,y \right)\) exist at \(\left(a,b \right)\), then \(\left(a,b \right)\) is a critical point of \(f \left(x,y \right)\) giving \(\nabla f \left(a,b \right) = \vec{0}\).

To prove this fact, we will use Fermat's Theorem:

Proof:
Let's define \(g \left(x \right) = f \left(x,b \right)\) given that \(f \left(x,y \right)\) has relative extrema at \(\left(a,b \right)\). That means \(g \left(x \right)\) has a relative extremum at \(x = a\), and by Fermat's Theorem, we can say that \(g' \left(a \right) = 0\) because \(x = a\) is the relative extrema. All of these facts mean that \(\nabla f \left(a,b \right) = \vec{0}\), and so \(f \left(x,y \right)\) has a critical point at \(\left(a,b \right)\).
Note that all critical points are NOT relative extrema. However, all relative extrema will be critical points of the function.

For example, let's say we have a surface in \(\mathbb{R}^3\):

\(f\left( {x,y} \right) = xy\)

Notice that the slope at the origin is \(0\) and is also a critical point. Does that mean the point is a maximum or a minimum? No. It will be called a saddle point as all four quadrants contain a higher value than \(f \left(0,0 \right)\).

So how can we determine if a critical point is a saddle point?

We can determine that by using the following facts:

Facts:

Suppose that \(\left(a,b \right)\) is a critical point of \(f \left(x,y \right)\) and that the second-order partial derivatives are continuous in some region that contains \(\left(a,b \right)\). Next, define, \(D = D \left(a,b \right) = f_{xx} \left(a,b \right) f_{yy} \left(a,b \right) - [f_{xy} \left(a,b \right)]^2\).

We then have the following classifications of the critical point:

  1. If \(D > 0\) and \(f_{xx} \left(a,b \right) > 0\) then there is a relative minimum at \(\left(a,b \right)\).
  2. If \(D > 0\) and \(f_{xx} \left(a,b \right) < 0\) then there is a relative maximum at \(\left(a,b \right)\).
  3. If \(D > 0\) then the point\(\left(a,b \right)\) is a saddle point.
  4. If \(D = 0\), the point \(\left(a,b \right)\) may be a relative minimum, maximum, or saddle point, other techniques need to be used to classify the critical point.


Keep in mind that if both \(D > 0\), then both \(f_{xx} \left(a,b \right)\) and \(f_{yy} \left(a,b \right)\) will have the same sign.
