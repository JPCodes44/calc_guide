Chapter 4.2 : Double Integrals

Reviewing the definition of definite integrals for single-variable functions is essential for discussing double integrals. Specifically, when dealing with integrals, we will use the following definition:

\(\displaystyle\int_{{\,a}}^{{\,b}}{{f\left( x \right)\,dx}}\)

But first, let's look at box areas for the first cases of double integrals.

\(\begin{array}{11}\displaystyle\iint_{{\,a,c}}^{{\,b,d}}{{f\left( x \right)\,dx dy}} \\ Or \\ \displaystyle\iint_{{\,a,c}}^{{\,b,d}}{{f\left( x \right)\,dx dy}}\end{array}\)

We can switch the order of \(d_y\) and \(d_x\) for these types of areas, as none of the bounds contain an independent variable.

But first, let's look at two general cases of double integrals that we need to consider before determining if we can switch dy and dx, given the following sketch:

For case 1 where the \(x\) boundaries are fixed and the \(y\) values are varied:

\(\displaystyle\iint\limits_{D}{{f\left( {x,y} \right)\,dA}} = \int_{{\,a}}^{{\,b}}{{\int_{{{g_{\,1}}\left( x \right)}}^{{{g_{\,2}}\left( x \right)}}{{f\left( {x,y} \right)\,dy}}\,dx}}\)

For case 2 where the \(y\) boundaries are fixed and the \(x\) values are varied:

\(\displaystyle\iint\limits_{D}{{f\left( {x,y} \right)\,dA}} = \int_{{\,c}}^{{\,d}}{{\int_{{h{\,_1}\left( y \right)}}^{{{h_{\,2}}\left( y \right)}}{{f\left( {x,y} \right)\,dx}}\,dy}}\)
