Chapter 4.3 : Double Integrals in Polar Coordinates

Reviewing the definition of definite integrals for single-variable functions is essential for discussing double integrals. For instance, let's suppose we wanted to do the following integral,

\(\displaystyle\iint\limits_{D}{{f\left( {x,y} \right)\,dA}},\,\,\,\,\,D{\mbox{ is the disk of radius 2}}\)

To describe this region in terms of \(x\) and \(y\), we must establish a set of inequalities for \(x\) and \(y\). These inequalities are:

\(\begin{array}{c} - 2 \le x \le 2\\ - \sqrt {4 - {x^2}} \le y \le \sqrt {4 - {x^2}} \end{array}\)

Setting up the double integral, we have:

\(\displaystyle\iint\limits_{D}{{f\left( {x,y} \right)\,dA}} = \int_{{\, - 2}}^{{\,2}}{{\int_{{ - \sqrt {4 - {x^2}} }}^{{\,\sqrt {4 - {x^2}} }}{{f\left( {x,y} \right)\,dy}}\,dx}}\)

As you can see, the integral in cartesian coordinates is a little bit aids, so we will use polar coordinates to make the process easier:

\(\begin{array}{c}0 \le \theta \le 2\pi \\ 0 \le r \le 2\end{array}\)

Note that we have replaced the \(dx\) and dy with \(d \theta\) and \(dr\). However, we cannot just replace them. Instead, we must use the jacobian or change of variables to obtain the proper conversion to polar:

\(dA = r\,dr\,d\theta\)

Recall from previous sections that converting from cartesian to polar and polar to cartesian uses the following formulas:

\(x = r\cos \theta \hspace{0.5in}y = r\sin \theta \hspace{0.5in}{r^2} = {x^2} + {y^2}\)
