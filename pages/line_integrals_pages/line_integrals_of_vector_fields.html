Chapter 5.4 : Line Integrals of Vector Fields

In this section, we will evaluate line integrals of vector fields instead of functions.

Let's suppose we have the following vector field:

\(\vec F\left( {x,y,z} \right) = P\left( {x,y,z} \right)\vec i + Q\left( {x,y,z} \right)\vec j + R\left( {x,y,z} \right)\vec k\)

And the three-dimensional, smooth curve is given by:

\(\vec r\left( t \right) = x\left( t \right)\vec i + y\left( t \right)\vec j + z\left( t \right)\vec k\hspace{0.25in}\hspace{0.25in}a \le t \le b\)

The line integral of \(\vec{F}\) along \(C\) is,

\(\displaystyle \int\limits_{C}{{\vec F\centerdot d\,\vec r}} = \int_{{\,a}}^{{\,b}}{{\vec F\left( {\vec r\left( t \right)} \right)\centerdot \vec r'\left( t \right)\,dt}}\)

Note that we have the dot product of the vector field where the differential is a vector. In fact, F(r(t)) represents:

\(\displaystyle \vec F\left( {\vec r\left( t \right)} \right) = \vec F\left( {x\left( t \right),y\left( t \right),z\left( t \right)} \right)\)

It is also possible to express line integrals of vector fields by using a line integral with respect to arc length. For example, this can be done in the following manner:

\(\displaystyle\int\limits_{C}{{\vec F\centerdot d\,\vec r}} = \int\limits_{C}{{\vec F\centerdot \vec T\,ds}}\)

Where \(T \left(t \right)\) is the unit tangent vector given by:

\(\vec T\left( t \right) = \dfrac{{\vec r'\left( t \right)}}{{\left\| {\vec r'\left( t \right)} \right\|}}\)

By subbing the unit tangent into the arc length integral, we get the same equation as before:

\(\displaystyle \begin{align*}\int\limits_{C}{{\vec F\centerdot d\,\vec r}} & = \int\limits_{C}{{\vec F\centerdot \vec T\,ds}}\\ & = \int_{{\,a}}^{{\,b}}{{\vec F\left( {\vec r\left( t \right)} \right)\centerdot \frac{{\vec r'\left( t \right)}}{{\left\| {\vec r'\left( t \right)} \right\|}}\,\,\left\| {\vec r'\left( t \right)} \right\|\,dt}}\\ & = \int_{{\,a}}^{{\,b}}{{\vec F\left( {\vec r\left( t \right)} \right)\centerdot \vec r'\left( t \right)\,dt}}\end{align*}\)

We use the first form to evaluate these line integrals because it is usually more straightforward.

To conclude this section, we will perform a general example to establish a helpful connection between line integrals of vector fields and line integrals with respect to \(x\), \(y\), and \(z\).

Given the following vector function \(\vec F\left( {x,y,z} \right) = P\,\vec i + Q\,\vec j + R\,\vec k\) and the curve \(C\) parameterized by \(\vec r\left( t \right) = x\left( t \right)\vec i + y\left( t \right)\vec j + z\left( t \right)\vec k\) the line integral is,

\(\displaystyle \begin{align*}\int\limits_{C}{{\vec F\centerdot d\,\vec r}} & = \int_{{\,a}}^{{\,b}}{{\left( {P\,\vec i + Q\,\vec j + R\,\vec k} \right)\centerdot \left( {x'\,\vec i + y'\,\vec j + z'\,\vec k} \right)\,dt}}\\ & = \int_{{\,a}}^{{\,b}}{{Px' + Qy' + Rz'\,dt}}\\ & = \int_{{\,a}}^{{\,b}}{{Px'\,dt}} + \int_{{\,a}}^{{\,b}}{{Qy'\,dt}} + \int_{{\,a}}^{{\,b}}{{Rz'\,dt}}\\ & = \int\limits_{C}{{P\,dx}} + \int\limits_{C}{{Q\,dy}} + \int\limits_{C}{{R\,dz}}\\ & = \int\limits_{C}{{P\,dx}} + Q\,dy + R\,dz\end{align*}\)

Therefore we have:

\(\displaystyle \int\limits_{C}{{\vec F\centerdot d\,\vec r}} = \int\limits_{C}{{P\,dx}} + Q\,dy + R\,dz\)

And don't forget about how reversing the direction of integration affects the integral:

\(\displaystyle \int\limits_{C}{{\vec F\centerdot d\,\vec r}} = \int\limits_{C}{{P\,dx}} + Q\,dy + R\,dz\)
