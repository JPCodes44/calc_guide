Chapter 5.5 : Fundamental Theorem of Line Integrals

Recall back to Calculus 1; we discovered the fundamental theorem of calculus that gave us a more straightforward way to integrate definite integrals:

\(\displaystyle \int_{{\,a}}^{{\,b}}{{F'\left( x \right)dx}} = F\left( b \right) - F\left( a \right)\)

For line integrals over specific vector fields, there is also a Fundamental Theorem of Line Integrals:

Suppose that C is a smooth curve given by r(t) between \(a \leq t \leq b\), and the gradient of f is continuous on C. Then,

\(\displaystyle \int\limits_{C}{{\nabla f\centerdot \,d\,\vec r}} = f\left( {\vec r\left( b \right)} \right) - f\left( {\vec r\left( a \right)} \right)\)

Given that \(r \left(a \right)\) represents the initial point on \(C\) while \(r \left(b \right)\) represents the final point on \(C\).

Proof:
Note that we can work in any dimension for this proof but will work in \(\mathbb{R}^3\) for now.

Let's start with evaluating the line integral:

\(\displaystyle \begin{align*}\int\limits_{C}{{\nabla f\centerdot d\,\vec r}} & = \int_{{\,a}}^{{\,b}}{{\nabla f\left( {\vec r\left( t \right)} \right)\centerdot \vec r'\left( t \right)\,dt}}\\ & = \int_{{\,a}}^{{\,b}}{{\left( {\frac{{\partial f}}{{\partial x}}\frac{{dx}}{{dt}} + \frac{{\partial f}}{{\partial y}}\frac{{dy}}{{dt}} + \frac{{\partial f}}{{\partial z}}\frac{{dz}}{{dt}}} \right)\,dt}}\end{align*}\)

We can then use Chain Rule to simplify the integrand:

\(\displaystyle \begin{align*}\int\limits_{C}{{\nabla f\centerdot d\,\vec r}} & = \int_{{\,a}}^{{\,b}}{{\left( {\frac{{\partial f}}{{\partial x}}\frac{{dx}}{{dt}} + \frac{{\partial f}}{{\partial y}}\frac{{dy}}{{dt}} + \frac{{\partial f}}{{\partial z}}\frac{{dz}}{{dt}}} \right)\,dt}}\\ & = \int_{{\,a}}^{{\,b}}{{\frac{d}{{dt}}\left[ {f\left( {\vec r\left( t \right)} \right)} \right]\,dt}}\end{align*}\)

To finish the proof, we know we can turn this type of integral into the definition for the Fundamental Theorem of Calculus:

\(\displaystyle \int\limits_{C}{{\nabla f\centerdot d\,\vec r}} = f\left( {\vec r\left( b \right)} \right) - f\left( {\vec r\left( a \right)} \right)\)

Note that we do not need to know the path we are integrating; we only need to know the initial and final points. Therefore changing the course will NOT change the value of the line integral.

As we work through line integrals over vector fields, we will need to define some new definitions and facts,

Definitions:

First suppose that \(\vec{F}\) is a continuous vector field in some region D.

  1. A vector field \(\vec{F}\) is considered conservative if there exists a function \(f\) that satisfies the equation \(\vec{F} = \nabla f\), where \(\nabla\) is the gradient operator. The function f is referred to as a potential function for the vector field. This definition was introduced in the first section of the current chapter.
  2. \(\displaystyle \int_C \vec{F} \centerdot \vec{dr}\) is independent-of-path if \(\displaystyle \int_{C_1} \vec{F} \centerdot \vec{dr} = \int_{C_2} \vec{F} \centerdot \vec{dr}\) for any two paths \(C_1\) and \(C_2\) in \(D\) have the same initial and final points.
  3. A path \(C\) is called closed if the initial starting points are the same as the final ones.
  4. A path \(C\) is simple if the path does not cross itself. For example, a figure 8 is not simple because the curve crosses itself.
  5. A region \(D\) is open if it doesn't contain any of it's boundary points.
  6. A region \(D\) is connected if we can connect any two points in the region \(D\) and have the path lie completely in \(D\).
  7. A region \(D\) is simply-connected if it is connected and contains no holes.

With these definitions, we can obtain the following facts,

Facts:

  1. \(\displaystyle \int\limits_{C}{{\nabla f\cdot \,d\,\vec r}}\) is independent of path
    To prove that the path taken doesn't affect the line integral, we just need to refer to a theorem stating that only the starting and ending points of a curve are required for the calculation.
  2. if \(\vec{F}\) is a conservative vector field then \(\displaystyle \int\limits_{C}{{\vec F\cdot \,d\,\vec r}}\) is independent of path.
    It's also simple to demonstrate the following fact: if a vector field \(\vec{F}\) is conservative, then it possesses a potential function f, and the line integral can be expressed as \(\displaystyle \int_{C} \vec{F} \cdot d \vec{r} = \int_{C} \nabla f \cdot d \vec{r}\). By utilizing the previous fact, we know that this line integral is independent of path.
  3. If \(\vec{F}\) is a continuous vector field on an open connected region \(D\) and if \(\displaystyle \int\limits_{C}{{\vec F\cdot \,d\,\vec r}}\) is independent of path (for any path in \(D\)) then \(\vec{F}\) is a conservative vector field on \(D\)
  4. If \(\displaystyle \int\limits_{C}{{\vec F\cdot \,d\,\vec r}}\) is independent of path then \(\displaystyle \int\limits_{C}{{\vec F\cdot \,d\,\vec r}} = 0\) for every closed path \(C\).
  5. If \(\displaystyle \int\limits_{C}{{\vec F\cdot \,d\,\vec r}} = 0\) for every closed path \(C\) then \(\displaystyle \int\limits_{C}{{\vec F\cdot \,d\,\vec r}}\) is independent of path.
