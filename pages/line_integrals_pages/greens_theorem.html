Chapter 5.7 : Green's Theorem

Let's determine the relationship between line integrals on closed paths and double integrals.

Recall that a closed curve is a curve \(C\) enclosing region \(D\).

Here is a sketch of what I mean:

Note that the curve is closed and straightforward, as no holes exist in region \(D\). Also note that curve \(C\) has a positive orientation if it is traced out in the counterclockwise direction.

Given curves/regions like this, we can use the following theorem:

Theorem

If a region \(D\) is enclosed by a positively oriented, piecewise smooth, simple, closed curve \(C\) and both \(P\) and \(Q\) possess continuous first-order partial derivatives within \(D\), then:

\(\displaystyle \int\limits_{C}{{Pdx\, + Qdy}} = \iint\limits_{D}{{\left( {\frac{{\partial Q}}{{\partial x}} - \frac{{\partial P}}{{\partial y}}} \right)\,dA}}\)

Here are some of the alternate notations that we will see in future problems assuming \(C\) satisfies the conditions of greens theorem:

\(\displaystyle \oint_{C}{{Pdx + Qdy}}\hspace{0.5in}{\mbox{or}}\hspace{0.25in} \mathop{\int\mkern-18.8mu \circlearrowleft}\limits_C {Pdx + Qdy}\)

So how would green's theorem work for regions with holes in them? Let's first split up the area into two parts \(D_1\) \(\cup\) \(D_2\) (recall that \(\cup\) represents union which represents \(D\) containing both \(D_1\) and \(D_2\)). Let's also define the boundary of \(D_1\), which is \(C_1\) \(\cup\) \(C_3\), and \(D_2\) is \(C_2\) \(\cup\) \(\left(-C_3\right)\). However, we know that \(C_3\) and \(-C_3\) will cancel, and we will be left with the equation on the far right.

We will begin by analyzing the given double integral and utilizing a fundamental property of double integrals to simplify it.

\(\displaystyle \iint\limits_{D}{{\left( {{Q_x} - {P_y}} \right)\,dA}} = \iint\limits_{{{D_1} \cup {D_2}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}} = \iint\limits_{{{D_1}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}} + \iint\limits_{{{D_2}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}}\)

Afterward, apply Green's theorem to each of these. Then, once more, use the property that allows us to separate line integrals into individual line integrals for each boundary segment.

\(\begin{align*}\iint\limits_{D}{{\left( {{Q_x} - {P_y}} \right)\,dA}} & = \iint\limits_{{{D_1}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}} + \iint\limits_{{{D_2}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}}\\ & = \oint\limits_{{{C_1} \cup {C_3}}}{{Pdx + Qdy}} + \oint\limits_{{{C_2} \cup \left( { - {C_3}} \right)}}{{Pdx + Qdy}}\\ & = \oint\limits_{{{C_1}}}{{Pdx + Qdy}} + \oint\limits_{{{C_3}}}{{Pdx + Qdy}} + \oint\limits_{{{C_2}}}{{Pdx + Qdy}} + \oint\limits_{{ - {C_3}}}{{Pdx + Qdy}}\end{align*}\)

Next, we'll use the fact that,

\(\displaystyle\oint\limits_{{ - {C_3}}}{{Pdx + Qdy}} = - \oint\limits_{{{C_3}}}{{Pdx + Qdy}}\)

It is important to recall that changing the orientation of a curve in line integrals with respect to x and/or y will change the sign on the integral. Using this fact, we arrive at the following:

\(\displaystyle \begin{align*}\iint\limits_{D}{{\left( {{Q_x} - {P_y}} \right)\,dA}} & = \oint\limits_{{{C_1}}}{{Pdx + Qdy}} + \oint\limits_{{{C_3}}}{{Pdx + Qdy}} + \oint\limits_{{{C_2}}}{{Pdx + Qdy}} - \oint\limits_{{{C_3}}}{{Pdx + Qdy}}\\ & = \oint\limits_{{{C_1}}}{{Pdx + Qdy}} + \oint\limits_{{{C_2}}}{{Pdx + Qdy}}\end{align*}\)

Putting the integrals back together, we get:

\(\displaystyle \begin{align*}\iint\limits_{D}{{\left( {{Q_x} - {P_y}} \right)\,dA}} & = \oint\limits_{{{C_1}}}{{Pdx + Qdy}} + \oint\limits_{{{C_2}}}{{Pdx + Qdy}}\\ & = \oint\limits_{{{C_1} \cup {C_2}}}{{Pdx + Qdy}}\\ & = \oint\limits_{C}{{Pdx + Qdy}}\end{align*}\)

What is the main takeaway from this exercise? Although it may be a lot of work, all we accomplished was to confirm the validity of Green's Theorem, which we already knew to be true. However, this exercise demonstrated that breaking up a region into smaller sections, as shown above, causes the line integrals on the parts of the curve that lie inside the region to cancel each other out (since they are moving in opposite directions). This concept will help deal with regions that have holes in them. To illustrate this point, consider a ring-shaped region:

Note how both curves are positively oriented because if we walk in the oriented direction, \(D\) will end up on our left side. However, you should also have noticed that the curve only has a positive orientation if counterclockwise. As we can see, \(C_2\) is moving in a clockwise direction. So why are both curves oriented positively? That's because, for regions with holes, there is an exception. Therefore, we will use the second definition as stated above.

This passage discusses how Green's Theorem can't be used to calculate line integrals over a curve that encloses a region with a hole. The curve in question is composed of two parts, \(C_1\) and \(C_2\). However, If we cut the disk in half and rename the portions of the curve, we obtain the following sketch:

Let's define the following boundaries for each of the domains:

\(D_1 \rightarrow C_1 \cup C_2 \cup C_5 \cup C_6\)

\(D_2 \rightarrow C_3 \cup C_4 \cup \left(-C_5 \right) \cup \left(-C_6 \right)\)

Now we can sub back the domains into the double integrals for the greens theorem because there are no holes in the domain:

\(\displaystyle \begin{align*}\iint\limits_{D}{{\left( {{Q_x} - {P_y}} \right)\,dA}} & = \iint\limits_{{{D_1}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}} + \iint\limits_{{{D_2}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}}\\ & = \oint_{{{C_1} \cup {C_2} \cup {C_5} \cup {C_6}}}{{Pdx + Qdy}} + \oint_{{{C_3} \cup {C_4} \cup \left( { - {C_5}} \right) \cup \left( { - {C_6}} \right)}}{{Pdx + Qdy}}\end{align*}\)

Breaking up the integral on each piece of the boundary, we get (note that \(C_5\) and \(C_6\) cancel because there's a negative and positive version of the curve on each integral):

\(\displaystyle \begin{align*}\iint\limits_{D}{{\left( {{Q_x} - {P_y}} \right)\,dA}} & = \iint\limits_{{{D_1}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}} + \iint\limits_{{{D_2}}}{{\left( {{Q_x} - {P_y}} \right)\,dA}}\\ & = \oint_{{{C_1}}}{{Pdx + Qdy}} + \oint_{{{C_2}}}{{Pdx + Qdy}} + \oint_{{{C_3}}}{{Pdx + Qdy}} + \oint_{{{C_4}}}{{Pdx + Qdy}}\end{align*}\)

Ultimately, we could have used Green's Theorem with regions containing holes.

We will now finalize this section by computing the area of a region \(D\) using Green's Theorem:

\(\displaystyle A = \iint\limits_{D}{{dA}}\)

To do this, we will think of the integral in terms of Green's Theorem. So let's suppose that:

\({Q_x} - {P_y} = 1\)

From here, let's determine some functions for \(P\) and \(Q\) that will satisfy this.

Here are some functions that will satisfy this:

\(\begin{array}{c}\begin{array}{l}P = 0 & & \\Q = x\end{array}&\begin{array}{l}P = - y & & \\\,\,Q = 0\end{array}&\begin{array}{l}P = - \frac{y}{2}\\\,\,Q = \frac{x}{2}\end{array}\end{array}\)

Using Green's Theorem, we can determine the area of the following region in \(D\):

\(\displaystyle A = \oint\limits_{C}{{x\,dy}} = - \,\oint\limits_{C}{{y\,dx}} = \frac{1}{2}\oint\limits_{C}{{x\,dy - y\,dx}}\)

Where \(C\) is the boundary of region \(D\).
