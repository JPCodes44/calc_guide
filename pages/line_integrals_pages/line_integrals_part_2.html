Chapter 5.3 : Line Integrals Part 2

The previous section dealt with line integrals relative to arc length. However, this section will focus on line integrals with respect to either x, y, or both. Similarly, we will begin with a two-dimensional curve C that has a specific parameterization:

Check which differential you have so you don't screw up, eh?

The line integral of \(f\) with respect to \(x\) is,

\(\displaystyle \int\limits_{C}{{f\left( {x,y} \right)\,dx}} = \int_{{\,a}}^{{\,b}}{{f\left( {x\left( t \right),y\left( t \right)} \right)x'\left( t \right)\,dt}}\)

The line integral of \(f\) with respect to \(y\) is,

\(\displaystyle\int\limits_{C}{{f\left( {x,y} \right)\,dy}} = \int_{{\,a}}^{{\,b}}{{f\left( {x\left( t \right),y\left( t \right)} \right)y'\left( t \right)\,dt}}\)

Remember integrals are also additive as follows:

\(\displaystyle \int\limits_{C}{{Pdx\, + Q\,dy}} = \int\limits_{C}{{P\left( {x,y} \right)dx}} + \int\limits_{C}{{Q\left( {x,y} \right)\,dy}}\)

Let's do an example:

What if we flip the direction of integration? Do we get the same value as before?

In this case, we get the same value except with a negative value. This pattern will always happen with these kinds of line integrals.

Let's define the following fact:

Fact

If \(C\) is any curve then,

\(\displaystyle \int\limits_{{ - C}}{{f\left( {x,y} \right)\,dx}} = - \int\limits_{C}{{f\left( {x,y} \right)\,dx}}\hspace{0.25in}\hspace{0.25in}{\rm{and}}\hspace{0.25in}\,\,\,\,\,\,\int\limits_{{ - C}}{{f\left( {x,y} \right)\,dy}} = - \int\limits_{C}{{f\left( {x,y} \right)\,dy}}\)

Combining the two integrals we get,

\(\displaystyle \int\limits_{{ - C}}{{Pdx\, + Q\,dy}} = - \int\limits_{C}{{Pdx\, + Q\,dy}}\)

It is also possible to perform integrals over three-dimensional curves. This involves adding a third integral with respect to the variable \(z\), resulting in a total of three integrals:

\(\begin{align*}\int\limits_{C}{{f\left( {x,y,z} \right)\,dx}} & = \int_{{\,a}}^{{\,b}}{{f\left( {x\left( t \right),y\left( t \right),z\left( t \right)} \right)x'\left( t \right)\,dt}}\\ \int\limits_{C}{{f\left( {x,y,z} \right)\,dy}} & = \int_{{\,a}}^{{\,b}}{{f\left( {x\left( t \right),y\left( t \right),z\left( t \right)} \right)y'\left( t \right)\,dt}}\\ \int\limits_{C}{{f\left( {x,y,z} \right)\,dz}} & = \int_{{\,a}}^{{\,b}}{{f\left( {x\left( t \right),y\left( t \right),z\left( t \right)} \right)z'\left( t \right)\,dt}}\end{align*}\)

Where the curve C is parameterized by,

\(x = x\left( t \right)\hspace{0.25in}y = y\left( t \right)\hspace{0.25in}z = z\left( t \right)\hspace{0.25in}a \le t \le b\)

Similar to the two-dimensional case, these three integrals will frequently appear together. Therefore, we will use a shorthand notation for them in this context.

\(\displaystyle \int\limits_{C}{{Pdx\, + Q\,dy + R\,dz}} = \int\limits_{C}{{P\left( {x,y,z} \right)dx}} + \int\limits_{C}{{Q\left( {x,y,z} \right)\,dy}} + \int\limits_{C}{{R\left( {x,y,z} \right)dz}}\)
