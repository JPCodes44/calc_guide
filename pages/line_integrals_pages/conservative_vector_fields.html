In general, solving potential functions goes through the following process:

  1. \(P\), \(Q\) and \(R\) are your partial derivatives; write them down (don't derivate them again!).
  2. Determine f by integrating either \(P\), \(Q\), or \(R\) with respect to the matching variable.
  3. Add the constant of integration as a function of the other variables not included in the differential.
  4. Differentiate the new function with respect to either one of the variables not included in the integrated differential.
  5. Set the differentiated function equal to the matching partial derivative, \(P\), \(Q\), or \(R\).
  6. Solve the constant and integrate with respect to whatever variable the partial derivative is with respect to.
  7. Add in the constant using similar rules as mentioned in step 3.
  8. Repeat step 6 for the other variable, except you find the value of the constant.
  9. Plug back the value of the constant into the final equation, and you're done!
