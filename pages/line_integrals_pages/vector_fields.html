Chapter 5.1 : Vector Fields

In math notation, a vector field can be defined as a function that assigns a vector to each point in a region of space. For example, let's say we have an area of space called D, a vector field F would be represented as:

\(F: D \rightarrow \mathbb{R}^3\)

Where \(\mathbb{R}^3\) is the three-dimensional real vector space and \(F \left(x,y,z \right) = f_1 \left(x,y,z \right), f_2 \left(x,y,z \right), f_3 \left(x,y,z \right)\) is a vector at the point \(\left(x,y,z \right)\) in \(D\).

Alternatively, we can also represent a vector field using differential operators. For example, if the vector field \(F\) is a differentiable function, we can write it as:

\(F = F_1 \left(x,y,z \right)\vec{i} + F_2 \left(x,y,z \right)\vec{j} + F_3 \left(x,y,z \right)\vec{k}\)

Where \(\vec{i}\), \(\vec{j}\), \(\vec{k}\) are the standard unit vectors in the \(x\), \(y\), and \(z\) directions, and \(F_1\), \(F_2\), and \(F_3\) are scalar functions that give the vector field's \(x\), \(y\), and \(z\) components.

We can then use differential operators such as the gradient \(\left(\nabla\right)\), divergence \(\left(div \right)\), and curl \(\left(curl \right)\) to analyze the behaviour of the vector field.

The final topic in this section is conservative vector fields. A vector field is called conservative if it can be expressed as the gradient of some function. This function is called a potential function. In other words, a vector field is conservative if it is also a gradient vector field for some function. For example, the vector field \(y \vec{i} + x \vec{j}\) is a conservative vector field with a potential function of \(f \left(x,y \right) = xy\).

However, the vector field \(-y \vec{i} + x \vec{j}\) is not conservative since there is no function \(f\) such that \(F = \nabla f\). The proof of this will be provided in a future section, along with a method for finding the potential function for a conservative vector field.
