In this chapter, we will take a look at coordinate systems in \(\mathbb{R}^3\). The concepts learned in this section will be very essential as most of the calculations done in Calc 3 is in 3 Dimensional space.

For Line integrals, you must become fluent in parameterizing a set of equations for a given curve in \(\mathbb{R}^3\) and \(\mathbb{R}^2\). If you have not mastered the subject yet, please go back and review parameterization.

In general, a line integral is defined by the parameterization:

\(x = h\left( t \right)\hspace{0.25in}y = g\left( t \right)\hspace{0.25in}\,\,\,\,a \le t \le b\)

Which is then put into a vector function:

\(\vec r\left( t \right) = h\left( t \right)\,\vec i + g\left( t \right)\vec j\hspace{0.25in}\hspace{0.25in}a \le t \le b\)

Within the interval \([a,b]\).

The line integral of \(f \left(x,y \right)\) along \(C\) is,

\(\displaystyle\int\limits_{C}{{f\left( {x,y} \right)\,ds}}\)

For defining the final equation of the line integral, recall the arc length of the interval from \(a,b\) was given the parameterization above:

\(\displaystyle L = \int_{{\,a}}^{{\,b}}{{ds}}\,\,,\hspace{0.25in}{\mbox{where }}ds = \sqrt {{{\left( {\frac{{dx}}{{dt}}} \right)}^2} + {{\left( {\frac{{dy}}{{dt}}} \right)}^2}} \,dt\)

We will use the arc length to define \(ds\) or, in other words, the differential surface.

Knowing this, we can form the final line integral:

\(\displaystyle\int\limits_{C}{{f\left( {x,y} \right)\,ds}} = \int_{{\,a}}^{{\,b}}{{f\left( {h\left( t \right),g\left( t \right)} \right)\sqrt {{{\left( {\frac{{dx}}{{dt}}} \right)}^2} + {{\left( {\frac{{dy}}{{dt}}} \right)}^2}} \,dt}}\)

However, we can simplify the following arc length equation regarding the vector function using the norm:

\(\displaystyle \sqrt {{{\left( {\frac{{dx}}{{dt}}} \right)}^2} + {{\left( {\frac{{dy}}{{dt}}} \right)}^2}} = \left\| {\,\vec r'\left( t \right)} \right\|\)

One thing we have yet to talk about regarding line integrals is the orientation at which we move across the boundaries. For example, we will determine an orientation for the direction of how the curve is being traced out. We want the initial bound to be \(A\) and the last \(B\). This boundary will be our positive orientation. Suppose we switch the order of \(A\) and \(B\) to flip the direction on its head. As a result, the line integral will now contain a negative instead of a positive sign due to the change in orientation.

Fact:

\(\displaystyle \int\limits_{C}{{f\left( {x,y} \right)\,ds}} = \int\limits_{{ - C}}{{f\left( {x,y} \right)\,ds}}\)

So far, we have only covered line integrals in \(\mathbb{R}^2\), but we can also extend the idea into \(\mathbb{R}^3\).

Let's suppose we have a three-dimensional curve \(C\) given by the parameterization:

\(x = x\left( t \right),\,\hspace{0.25in}y = y\left( t \right)\hspace{0.25in}z = z\left( t \right)\hspace{0.25in}a \le t \le b\)

Everything pretty much stays the same, except that we define another component as follows:

\(\displaystyle \int\limits_{C}{{f\left( {x,y,z} \right)\,ds}} = \int_{{\,a}}^{{\,b}}{{f\left( {x\left( t \right),y\left( t \right),z\left( t \right)} \right)\sqrt {{{\left( {\frac{{dx}}{{dt}}} \right)}^2} + {{\left( {\frac{{dy}}{{dt}}} \right)}^2} + {{\left( {\frac{{dz}}{{dt}}} \right)}^2}} \,dt}}\)

Or

\(\displaystyle \int\limits_{C}{{f\left( {x,y,z} \right)\,ds}} = \int_{{\,a}}^{{\,b}}{{f\left( {x\left( t \right),y\left( t \right),z\left( t \right)} \right)\displaystyle \left\| {\,\vec r'\left( t \right)} \right\| \,dt}}\)

Given the vector function:

\(\vec r\left( t \right) = \left\langle {x\left( t \right),y\left( t \right),z\left( t \right)} \right\rangle\)
